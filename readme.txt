== Description ==

Appointment Booking Scheduler WordPress plugin can book and manage appointments for you in your WordPress website through very easy steps. It provide the number of options and settings to manage your booking system


**Benefits Of Online Booking System**

Nowadays, more and more people benefit from online solutions that offer booking services. It is fast and convenient way to control booking process with minimum efforts and maximum result. Appointment scheduler is designed to automate booking process and introduce easy way to manage your booking from any desktop or mobile device you have. Having calendar with real-time availability can become a significant promotion of your business and may encourage clients to book online services right from your website.

== Installation ==

**Appointment Booking Scheduler [Installation Guide]**

1. You can:
 * Upload the 'appointment-booking-scheduler' folder to '/wp-content/plugins/' directory via FTP.
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **Appointment Booking Scheduler** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Ready, now you can enjoy it, and if you like it and find it useful.

**How to use Appointment Booking Scheduler?**

1. Download the plugin using WordPress.org or you can download the plugin through WordPress admin dashboard 'Plugins -> Add New'
2. After download and activate plugin, go to 'Appointment Booking Scheduler' tab on WordPress Dashboard.
3. Add **[abs_booking]** shortcode on any Page or Post to display booking form on the website. The user will select the service and time accoding to his/her need and then fill his/her detail and confirm the appointment for booking. After the successfull booking both the administrator and client will receive the pending notification through e-mail. Admin will go to the plugin dashboard and change the status of appointment from pending to approve or cancel from back-end and the client will receive the appointment confirmation or cancellation notification.
